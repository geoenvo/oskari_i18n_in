Panduan pengguna ini berisi informasi tentang jendela peta dan fungsinya. Di jendela peta Anda dapat menyusun tampilan peta dan menelusuri kumpulan data geografis dengan membuka layer peta satu per satu atau beberapa sekaligus dengan tingkatan transparansi yang berbeda-beda. Anda juga dapat menyematkan peta di situs web Anda sendiri atau membagikan tautan dengan pengguna eksternal. Panduan ini berisi informasi tentang cara kerja jendela peta. Selain itu jendela peta juga menyediakan panduan interaktif untuk berbagai situasi penggunaan.

Cara menggunakan peta

Sebuah peta terdiri dari berbagai layer peta yang disediakan oleh para penyedia service. Hal pertama yang Anda lihat saat membuka jendela peta adalah tampilan default. Anda dapat menghapus peta ketika diperlukan dengan mengeklik Restore defaults yang terdapat di Tools pada kiri bawah jendela, atau di tengah kontrol panning.

 

Fungsi dasar

Geser peta dengan mengeklik and menyeretnya atau dengan cara mengeklik kontrol panning yang terdapat di kanan atas jendela. Gunakan fungsi Pencarian jika Anda mencari obyek atau lokasi tertentu.

Perbesar atau perkecil peta dengan mengeklik tombol + dan - di ujung kontrol skala, atau klik dan geser kontrol skala untuk menentukan skala yang diinginkan.

Koordinat menunjukkan lokasi pointer mouse pada peta.

Peta Index menampilkan view peta dalam skala yang lebih kecil. Anda dapat menampilkan peta index dengan mengeklik di kanan bawah jendela peta.

Obyek yang terdapat pada peta seringkali memiliki informasi tambahan. Klik obyek pada peta untuk melihat informasi tersebut. Produk data ditampilkan dalam sebuah daftar yang berisi semua obyek yang berada dalam area yang ditampilkan pada jendela peta. Anda dapat menentukan kondisi untuk membatasi pilihan obyek jenis tertentu dalam sebuah data produk. Daftar yang berisi informasi mengenai suatu obyek akan tampil ketika Anda mengeklik tautan data obyek pada sisi kanan peta.

 

Menu

Gunakan menu Pencarian untuk mencari sebuah lokasi dan memusatkan peta pada lokasi tersebut.

Gunakan menu Layer Peta untuk mencari dan memilih dataset dari berbagai penyedia data. Dataset yang tampil disortir menurut tema dan penyedia data. Dataset:

Simbol abu-abu berarti dataset tersebut tidak dapat ditampilkan pada tingkat skala yang terpilih. Saat Anda memilih produk data, sebuah tautan data obyek akan tampil pada sisi kanan peta. Klik tautan tersebut untuk melihat daftar data terkait obyek yang ditampilkan pada peta.

Gunakan menu Layer Terpilih untuk melihat dataset yang ditampilkan pada view peta. Anda dapat menyesuaikan urutan dan transparasi dari layer yang ada pada peta ataupun menghapusnya. Lingkaran di kanan menu menunjukkan jumlah layer peta yang terpilih.

Gunakan menu Data Saya untuk melihat semua obyek, view (lihat bagian Tools di bawah) yang tersimpan, serta peta yang Anda pernah sematkan. Di sini Anda juga dapat membatalkan penyematan suatu peta. Pada menu ini juga terdapat informasi pengguna Anda yang dapat dirubah. Anda harus login dahulu untuk menggunakan fungsi ini.

Gunakan menu Publikasi Peta untuk membuat kode view peta yang telah Anda susun, yang kemudian dapat disematkan pada situs web Anda maupun perusahaan. Anda harus login dahulu untuk menggunakan fungsi ini.

Menu Legenda Peta berisi deskripsi dari simbol-simbol yang digunakan pada layer peta.

Menu Panduan Pengguna berisi panduan pengguna ini. Jika Anda memerlukan bantuan, simbol i atau tautan informasi selengkapnya dapat memberikan informasi tambahan.
Harap diingat kalau Anda selalu dapat mereset setelan default atau melakukan undo maupun redo perubahan yang dilakukan. Tool tersebut berada di sisi kiri, di bawah bagian menu (lihat Tools).

 

Tools

- Perbesar area terpilih | Geser peta

View peta

Buat view peta Anda dengan memilih layer peta pada menu Layer Peta atau dari layer peta yang Anda simpan di daftar Layer Peta Saya. Gunakan menu Layer Terpilih untuk mengatur urutan layer peta Anda dengan cara mengeklik dan menyeret serta merubah transparasi dari layer peta.

Cara menyusun view peta:
1.    Pilih layer peta yang ingin Anda tampilkan pada menu Layer Peta
2.    Gunakan menu Layer Terpilih untuk mengatur urutan layer peta Anda dengan cara mengeklik dan menyeret serta merubah transparasi dari layer peta jika diperlukan.

Cara menyimpan view peta
Gunakan tool Simpan untuk menyimpan view peta. Informasi mengenai layer peta yang terpilih, urutan serta transparansinya, lokasi terpilih, dan tingkat skala akan disimpan di View Peta Saya. Daftar view peta yang tersimpan ada di tab View Peta Saya pada menu Data Saya.

Cara membagikan view peta
Tool Tautan ke View Peta dapat membuat tautan langsung ke view peta yang telah disusun. Anda dapat menggunakannya untuk membagikan view peta dengan pengguna Internet lainnya. Harap diingat bahwa untuk saat ini Anda belum dapat membagikan lokasi maupun layer peta milik Anda sendiri.

Cara membuat dan menyematkan view peta

Anda dapat menyusun sebuah view peta dan membuatnya menjadi publik dengan cara menyematkannya melalui proses copy paste beberapa baris kode html. Hal ini dapat dilakukan melalui menu Publikasi Peta. Anda harus login dahulu untuk menggunakan fitur ini.

Untuk membuat sebuah peta Anda cukup memberikan judul untuk peta tersebut, tentukan ukuran peta dan jenis URL untuk situs web tujuan tempat peta akan disematkan. Anda juga dapat memilih tool apa saja yang akan ditampilkan pada peta seperti kontrol skala dan pencarian nama lokasi.

Terakhir, Anda akan mendapat keluaran beberapa baris kode html. Copy paste kode tersebut ke laman web tujuan di mana peta hendak ditampilkan.

 

Obyek/Lokasi Saya

Anda dapat menambahkan obyek/lokasi milik Anda ke peta. Sebuah obyek dapat berupa titik, garis, maupun area dengan nama dan deskripsi. Saat Anda menandai sebuah obyek pada peta, klik "Simpan" untuk menyimpannya ke salah satu layer peta Anda atau ke sebuah layer peta yang baru. Anda dapat menggunakan layer peta yang berbeda-beda untuk melihat obyek-obyek yang memiliki tema yang sama. Anda harus login dahulu untuk dapat menyimpan, mengedit, dan menghapus obyek milik Anda.

Di manakah Lokasi Saya?
Obyek/lokasi yang Anda simpan terdapat di menu Data Saya pada tab Lokasi.

Cara melihat Lokasi Saya:
Klik nama sebuah obyek yang terdapat di tab Lokasi untuk melihatnya.

Cara membagikan Lokasi Saya:
Untuk saat ini belum dimungkinkan untuk dapat membagikan tautan ke obyek/lokasi yang telah Anda simpan.

Cara mengedit Lokasi Saya:
Gunakan fungsi Edit untuk merubah nama, deskripsi atau layer peta dari obyek/lokasi yang tersimpan.

Cara menghapus Lokasi Saya:
Gunakan fungsi Hapus untuk menghapus obyek/lokasi yang tersimpan dari layer peta.

 

Layer Peta Saya

Anda dapat menyimpan obyek/lokasi pada layer peta Anda. Gunakan layer peta yang berbeda untuk membedakan antara satu tema dengan yang lainnya. Anda dapat menemukannya dengan memilih Layer Peta Saya dari menu drop-down di tab Lokasi pada menu Data Saya.

Cara membuat layer peta baru:
Saat Anda menyimpan atau mengedit sebuah obyek/lokasi, secara otomatis Anda telah membuat sebuah layer peta. (Anda harus login dahulu untuk menyimpan atau mengedit.)

Cara melihat Layer Peta Saya:
Anda dapat melihat layer peta Anda yang tersimpan ketika Anda mengeklik sebuah obyek pada daftar di tab Lokasi pada menu Data Saya atau dengan mencentang layer yang diinginkan di bawah kategori Lokasi Saya pada menu Layer Peta.

Cara mengedit dan menghapus Layer Peta Saya:
Anda dapat mengedit nama atau style dari layer peta dengan mengeklik pilihan Edit.

Setelan privasi dari Layer Peta Saya:
Layer Peta Saya dan semua obyek/lokasi di dalamnya sudah privat secara default.

Cara membuat Layer Peta Saya menjadi publik:
Untuk saat ini belum dimungkinkan untuk membuat layer peta buatan pengguna menjadi publik.
